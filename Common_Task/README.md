The task is a three-class classification of simulated strong-lensing images: no substructure, subhalo substructure (spherical WIMP-like halos), and vortex substructure (a signature of superfluid dark-matter condensates). Images are single-channel, 150×150 pixels, normalised with min-max scaling.
The sphere class is underrepresented, which causes naive classifiers to under-recall it.

The final model weights are also uploaded as a .pth file, along with all the output images, for easier viewing along with the IPYNB notebook.
