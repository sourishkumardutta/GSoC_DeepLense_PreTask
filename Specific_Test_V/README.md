This task uses real observational data — galaxy images structured as train_lenses / train_nonlenses and test_lenses / test_nonlenses. 
Each object is a 3-channel (g, r, i filter) image of shape (3, 64, 64). The defining challenge is severe class imbalance: roughly 100× more non-lenses than lenses. 
This is not just a machine learning nuisance — it reflects the fundamental reality of automated lens searches.
In real surveys, lenses occur at roughly 1 per 1,000–10,000 objects, so even a 1% false positive rate generates an overwhelming number of false positives that human reviewers cannot practically process. 
Understanding this drove every choice I made in my solution.
